# 代码版多邻国：人工智能训练师实操学习工具

这是一个零依赖 Web App/PWA 原型，用来验证“把人工智能训练师实操题拆成多邻国式微学习任务”的体验。

## 已实现

- 首页 Dashboard：今日 XP、连续学习天数、生命值。
- 时间入口：30 秒、2 分钟、5 分钟、10 分钟。
- 题型：选择题、填空题。
- 题库规则：主线题只从 `.ipynb` 的下划线空位生成填空题；选择题只用于少量“函数选择”。
- 展示规则：每题必须显示来自 `.ipynb` 注释的中文任务提示，再显示原题代码空位。
- 四层反馈：能否运行、测试结果、更好写法、知识点总结。
- 技能树：按 XP 显示三星掌握等级。
- 复习中心：错题自动进入复习队列，显示学习热力图。
- 题库管理：展示本地实操题来源、原始题和 AI 拆题预览。
- PWA：包含 manifest 和 service worker，可后续添加到桌面。

## 题库来源

MVP 样例题来自本地人工智能训练师实操题库和 Obsidian 笔记：

- `人工智能训练师三级考试平台模拟界面/1.1.1.html`
- `人工智能训练师三级考试平台模拟界面/1.1.2_20250716_135623.html`
- `人工智能训练师三级考试平台模拟界面/1.1.3.html`
- `人工智能训练师三级上网素材/`
- `08_实操题代码笔记/`

题目数据放在 `questions.json`，页面不硬编码题目。

## 本地运行

在当前目录执行：

```bash
npm run check
npm start
```

然后打开：

```text
http://localhost:4173
```

建议通过本地服务打开。因为页面会读取 `questions.json`，直接双击 `index.html` 时，部分浏览器会拦截本地文件读取。
